## ⚡ I Also Rescue Broken AI-Generated MVPs
A growing share of my 2026 work is fixing products built on Lovable, Bolt, v0, Cursor, and raw LLM output. Typical findings when I take over:

| Common Issue | What It Costs You |
|---|---|
| 🔴 &nbsp; API integrations that silently drop payloads | Lost orders, ghost users, support fires |
| 🔴 &nbsp; Auth missing tenant isolation | Any user can read any row, GDPR exposure |
| 🔴 &nbsp; Stripe webhooks that "work" until a refund | Reconciliation nightmare and chargebacks |
| 🔴 &nbsp; Bloated, copy-pasted code | Every new feature breaks three old ones |
| 🔴 &nbsp; No migrations, audit logs, or rollback path | Production is one bad deploy from data loss |
| 🔴 &nbsp; Hidden security holes the founder cannot see | Breach risk, investor red flag, churn |

I rebuild these into products that can onboard real customers without leaking data or burning support hours.
